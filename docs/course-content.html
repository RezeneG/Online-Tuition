Understanding Text Elements

HTML provides several elements for working with text content. Let's explore the most commonly used ones:

index.html
<!DOCTYPE html> <html> <head> <title>Text Elements Example</title> </head> <body> <h1>Main Heading</h1> <h2>Subheading</h2> <p>This is a paragraph of text.</p> <strong>This text is bold</strong> <em>This text is italic</em> </body> </html>

Working with Links

Links are created using the <a> (anchor) element. They can link to other pages, files, or specific sections within a page.

links.html
<!DOCTYPE html> <html> <head> <title>Links Example</title> </head> <body> <!-- External link --> <a href="https://www.example.com">Visit Example.com</a> <!-- Internal link --> <a href="about.html">About Us</a> <!-- Link with target --> <a href="https://www.example.com" target="_blank"> Open in new tab </a> <!-- Email link --> <a href="mailto:contact@example.com">Email Us</a> </body> </html>
